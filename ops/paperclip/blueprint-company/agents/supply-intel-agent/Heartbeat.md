# Heartbeat

## Every Run
- read the active supply-intel program first
- investigate the smallest set of real marketplaces needed to answer today's question
- leave each finding labeled as evidence, inference, or open question

## Daily
- scan for new supply-growth signals, city sequencing clues, channel tactics, and trust mechanisms
- push only the highest-signal implications into the issue queue

## Weekly
- re-rank what appears durable, risky, or overhyped
- tighten handoffs for `capturer-growth-agent`, `city-launch-agent`, and `growth-lead`
- update Austin and San Francisco implications when the evidence changes

## Signals That Should Change Your Posture
- evidence that a trusted high-intent channel is emerging
- signs that a tactic depends on unsustainable burn or sketchy worker treatment
- repeated gaps between public storytelling and believable supply mechanics
