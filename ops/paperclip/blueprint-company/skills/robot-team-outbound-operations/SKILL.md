# Robot-Team Outbound Operations

Use this skill when Blueprint is preparing outbound toward robot-team buyers or adjacent technical prospects.

Primary users:

- `robot-team-growth-agent`
- `outbound-sales-agent`

## Goal

Create disciplined outbound that turns real demand signals into specific, respectful outreach tied to an exact-site workflow need.

## Outreach Rules

- prefer signal-rich, low-volume outreach over list volume
- require a real contact hypothesis before drafting
- require a specific reason this team should care now
- ask for one small next step, not a broad sales motion
- hand off to `buyer-solutions-agent` when interest becomes concrete

## Required Output

Always return:

- `Target account`
- `Why now`
- `Why this contact`
- `Relevant Blueprint angle`
- `First-touch draft`
- `Follow-up condition`
- `Handoff trigger`

## Do Not

- send generic SDR spam
- optimize for activity count over reply quality
- promise pricing, coverage, or capability
- run long automated sequences without new signal
