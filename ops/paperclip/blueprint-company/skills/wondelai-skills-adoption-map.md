# Wondelai Skills Adoption Map

Reference repo:

- `ops/paperclip/external/wondelai-skills/`

This repo is vendored as source material only. Do not bind its upstream `SKILL.md` files directly to Blueprint agents.

Use Blueprint wrapper skills so the imported frameworks stay aligned with:

- capture-first, world-model-product-first doctrine
- exact-site package and hosted-review truth
- Paperclip-as-work-record execution
- no invented traction, guarantees, supply, or capability

## Explicit Non-Adoption

Do not bind `wondelai` source skills directly to:

- executive routing agents
- ops execution agents
- rights, privacy, or procurement gates
- engineering implementation and review agents

Those roles need Blueprint-native operational and truth-boundary skills, not generic business frameworks.
